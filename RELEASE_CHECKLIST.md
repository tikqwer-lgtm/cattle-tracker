# Чек-лист перед релизом

Перед каждым релизом рекомендуется вручную проверить ключевые сценарии.

## Вход и выход

- [ ] Открывается экран входа (логин, пароль, кнопки «Войти», «Регистрация», «Без авторизации»).
- [ ] Вход по логину и паролю (локально или с сервером) переводит на главное меню.
- [ ] Кнопка «Выйти» на главном экране выполняет выход и возвращает на экран входа.
- [ ] После выхода фокус попадает в поле пароля (или логина).

## Главное меню и навигация

- [ ] Все группы меню открываются: «Работа с данными», «Действия», «Аналитика», «Уведомления и планы», «Настройки».
- [ ] В подменю кнопки ведут на соответствующие экраны (добавить животное, список, осеменение, отёл, УЗИ, аналитика, протоколы и т.д.).
- [ ] Переключатель объектов (баз) отображается и работает: выбор базы, «Изменить», «Удалить», «+ Добавить объект».

## Данные

- [ ] «Список всех животных» показывает записи (при наличии данных; при 0 записей — сообщение «Нет записей»).
- [ ] «Все осеменения» показывает данные об осеменениях (при наличии).
- [ ] Информационное табло на главном экране (всего коров, стельных, в сухостое и т.д.) совпадает с количеством записей в списке.
- [ ] Добавление новой записи (корова) сохраняется и отображается в списке.
- [ ] Редактирование записи сохраняется.
- [ ] Удаление записи работает.

## Протоколы и настройки

- [ ] Экран «Настройки — Протоколы синхронизации» открывается и показывает список протоколов и форму добавления/редактирования.
- [ ] Добавление, редактирование и удаление протокола работают (локально или с сервером при включённом API).

## Синхронизация с сервером

- [ ] При включённом адресе сервера данные загружаются с API.
- [ ] Переключение базы (объекта) подгружает записи и протоколы для выбранной базы.
- [ ] Экспорт/импорт базы (если реализован в UI) включает протоколы.

## Экспорт и импорт

- [ ] Экспорт в Excel (или выгрузка списка) работает.
- [ ] Импорт из файла (CSV/Excel) при необходимости работает.

## Десктоп (Electron)

- [ ] Приложение запускается через `npm run electron` (или установленный exe).
- [ ] После сборки `npm run electron:dist` установщик из `electron/dist/` устанавливается и запускается без ошибок.
